# GreenLearnAI — план развития платформы

> Видение: ребёнок делает реальное доброе дело → загружает фото → получает GreenPoints →
> обменивает на реальную награду. Телефон — только для отчёта, жизнь — в реальном мире.

## Этап 0 — Ядро (готово)

- [x] Регистрация/авторизация родителей (JWT)
- [x] Профили детей (2–17 лет), уровни, серии дней
- [x] Миссии по категориям, фото-отчёты, AI-проверка фото, ручная проверка родителем
- [x] GreenPoints: начисление, достижения
- [x] AI-чат: текст + голос (Web Speech API), озвучка ответов
- [x] Режим «Личный психолог» с правилами безопасности и кризисными ответами
- [x] Возрастные персонажи: Жашыл (≤12), GAIA (13–17), GAIA Pro (взрослые)
- [x] Магазин наград: каталог, обмен GP на купоны с кодами, история
- [x] Голосовой дневник с AI-анализом прогресса речи
- [x] Eco Passport: сводка добрых дел + автоматические сертификаты + PDF

## Этап 1 — Укрепление ядра (1–2 месяца)

**Цель: продукт, который не стыдно показать школе и инвестору.**

1. **Кыргызский язык интерфейса** — i18n на фронтенде (react-i18next), переключатель RU/KG.
2. **Анимированный Жашыл** — визуальный персонаж в чате (Lottie-анимация), реакции на события.
3. **AI Story Mode** — Жашыл начинает историю, ребёнок продолжает голосом, бот задаёт
   вопросы «почему герой так поступил?» (новый режим чата, переиспользует голосовой ввод).
4. **Миссии против телефонной зависимости** — таймерные миссии (30 мин/вечер/прогулка
   без телефона) с подтверждением родителем.
5. **Семейные миссии** — флаг `is_family` у миссии, повышенные баллы, участвуют все дети.
6. **Push/email-напоминания** — «сегодня ещё нет записи в дневнике», «серия под угрозой».

## Этап 2 — Сообщество и соревнование (2–4 месяца)

**Цель: вирусный рост через школы.**

1. **Роли и организации** — модели School, ClassRoom, Teacher; директор распределяет
   пакеты GreenPoints по классам, учитель создаёт свои условия-миссии.
2. **Школьный рейтинг** — таблица лидеров школ по GreenPoints, сезоны, реальный приз.
3. **Эко-карта Бишкека** — Leaflet/MapLibre, отметки «здесь убрали мусор / посадили дерево»,
   привязка к одобренным миссиям с геотегом фото.
4. **Профиль достижений публичный** — шарабельная страница Eco Passport (ссылка для школы).

## Этап 3 — Монетизация (4–6 месяцев)

1. **B2B-кабинет школ** — покупка пакетов GP, аналитика по классам, выгрузка отчётов.
2. **Брендовые миссии** — кабинет партнёра (KFC Green Mission и т.п.), модерация,
   статистика выполнений для бренда.
3. **Premium для родителей** — расширенная аналитика: динамика речи из дневника,
   баланс категорий (эко/учёба/здоровье/добро/творчество), недельные отчёты.
4. **Платёжная интеграция** — локальные платёжные системы КР (Mbank, O!Деньги, Элсом).

## Этап 4 — Масштаб (6–12 месяцев)

1. Мобильное приложение (React Native / Expo) — пуши, камера, офлайн-черновики.
2. Серверное распознавание речи (Whisper) — точный кыргызский, независимость от браузера.
3. Распознавание прогресса речи метриками (длина предложений, словарь) — графики в Premium.
4. Экспансия: Ош, Каракол; затем Казахстан/Узбекистан.
